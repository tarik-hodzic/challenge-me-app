<h1>ChallengeMe – Daily Challenge Android App</h1>

ChallengeMe is an Android application developed in Kotlin that motivates users through daily and random challenges.  
The app focuses on habit building, streak tracking, and maintaining user engagement through a clean and modern interface.

<h2>Technologies Used</h2>

- Kotlin  
- Android SDK  
- Jetpack Compose  
- Android Notifications  

<h2>Project Purpose</h2>

This project was built to:

- Develop an Android application using Kotlin  
- Apply modern Android development concepts with Jetpack Compose  
- Implement state handling and user interaction logic  
- Design a challenge-based motivation and reward system  

<h2>Features</h2>

- Daily challenges that refresh automatically  
- Random challenges with higher reward value  
- Streak system for tracking completed challenges  
- Point-based reward system  
- Challenge completion tracking  
- Persistent state handling within the app  
- Clean and minimal user interface  

<h2>App Logic Overview</h2>

- Daily challenges are generated and refreshed once per day  
- Random challenges provide additional points  
- Completed challenges are stored and added to the user streak  
- The streak screen displays completed challenges and total points  
- Notifications can be used to remind users to complete challenges  

<h2>How to Run the Project</h2>

1. Open the project in Android Studio  
2. Allow Gradle to sync dependencies  
3. Run the app on an Android emulator or physical Android device  

<h2>Project Scope</h2>

This project focuses on core Android development concepts, including UI composition, state management, and user interaction.
